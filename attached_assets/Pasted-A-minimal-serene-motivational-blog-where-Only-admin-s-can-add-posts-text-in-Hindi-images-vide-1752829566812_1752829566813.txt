A minimal, serene motivational blog where:

Only admin(s) can add posts (text in Hindi, images, videos).

Public users can view posts, filter by host/creator, and toggle dark/light mode.

Designed to be clean, distraction-free, and accessible to all age groups.

🌐 Website Pages (React Routes)
Page	Path	Description
Home	/	Landing page with welcome message, recent motivational post(s), dark/light toggle
All Posts	/posts	Public feed of all motivational posts (image/video/text), filter by creator
View Post	/posts/:postId	Detailed view of a single post with content, media, and author info
Login (Admin only)	/admin/login	Secure login page for admin/host
Admin Dashboard	/admin/dashboard	Add, edit, delete posts (only for admin); view stats
About	/about	Info about the blog’s mission, authors
Contact	/contact	Basic contact form for feedback/messages

✍️ Post Types (to be handled in UI and backend)
Each post must support:

Hindi Text Content

Image Upload (with optional caption)

Video Upload/YouTube Embed

🔐 User Roles
Role	Access
Admin/Host	Add/Edit/Delete posts, login dashboard
Public/Guest	View all posts, filter by creator

You can limit writing access by checking the user.role field in Firebase before allowing actions.
